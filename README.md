# learn inistallation

1- For quick installation, type the following command to install the script prerequisites.<br>
```
  bash <(curl -Ls https://raw.githubusercontent.com/MrAminiDev/Nexo-SelfBot/main/install.sh)
```

2- Then edit the contents of the file with the command below ( Line  20 ~ 22 )
```
nano main.py
```
3- To run the script, press <br>
```
nohup python3 main.py &
```
4- You can use the following command to turn off the robot
```
pkill -f "main.py"
```
